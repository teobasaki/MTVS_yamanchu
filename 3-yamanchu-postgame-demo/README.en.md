# Yamanchu Post-Game Demo

Standalone presentation demo for the post-game experience: date-course recommendations, AI course consultation, meetup community, and mock payment.

## Run

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

## Demo Flow

1. Review recommended courses and matching data in the post-game hub.
2. Compare places, pricing, sync score, and menus in the date-course view.
3. Use AI course consultation to review conversational recommendations and select a course.
4. Browse meetup communities with similar cheering/personality styles.
5. Complete a mock course payment.

## Scope

- Frontend-only demo
- No backend
- No database
- No real payment
- No real AI API call
