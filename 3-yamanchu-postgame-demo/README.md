# 야만추 경기 후 데모

경기 종료 후 데이트 코스 추천, AI 코스 상담, 정모 커뮤니티, 결제 Mock을 보여주는 발표용 단독 데모입니다.

## 실행

```bash
npm install
npm run dev
```

빌드:

```bash
npm run build
```

## 시연 흐름

1. 경기 후 허브에서 추천 코스와 매칭 데이터를 확인합니다.
2. 데이트 코스 화면에서 코스별 장소, 가격, 동조율, 메뉴를 비교합니다.
3. AI 코스 상담에서 대화형 추천을 확인하고 코스를 선택합니다.
4. 정모 커뮤니티에서 같은 응원 성향의 모임을 확인합니다.
5. 코스 결제를 Mock으로 완료합니다.
