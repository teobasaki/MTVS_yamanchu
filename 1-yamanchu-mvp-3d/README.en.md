# Yamanchu MVP 3D Demo

Standalone desktop presentation demo for the core Yamanchu MVP.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5173` or the port shown by Vite.

## Demo Flow

Landing -> 5 onboarding questions -> gold personality card -> 3D baseball stadium -> throw ball / swing bat -> NPC profile -> balance game -> sync score -> chat -> game ticket suggestion / mock payment

## Controls

- `W` / `Arrow Up`: move forward
- `S` / `Arrow Down`: move backward
- `A` / `Arrow Left`: rotate left
- `D` / `Arrow Right`: rotate right
- Right-side buttons: throw ball / swing bat

## Scope

- No live multiplayer
- No backend
- No database
- No real payment
- No GPT API
- NPC-based demo flow
