# 야만추 MVP 3D Demo

데스크탑 발표용 단독 시연 MVP입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 또는 표시된 포트로 접속합니다.

## 구현 플로우

랜딩 → 질문 5개 → 골드 성향 카드 발급 → 3D 야구장 → 공 던지기/배트 휘두르기 → NPC 프로필 → 밸런스 게임 → 동조율 계산 → 채팅 → 직관 제안/결제 Mock

## 조작

- W / ↑ : 전진
- S / ↓ : 후진
- A / ← : 왼쪽 회전
- D / → : 오른쪽 회전
- 화면 오른쪽 버튼: 공 던지기 / 배트 휘두르기

## 구현 범위

- 실시간 멀티플레이 없음
- 백엔드 없음
- DB 없음
- 실제 결제 없음
- GPT API 없음
- NPC 기반 데모
