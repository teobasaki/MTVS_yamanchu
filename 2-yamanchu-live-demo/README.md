# 야만추 직관 응원석 데모

내일 발표용 단독 데모입니다.

## 실행

```bash
npm install
npm run dev -- --port 5183
```

빌드:

```bash
npm run build
```

## 시연 흐름

1. 헤더의 [다음 상황] 버튼으로 타임라인을 진행시킵니다.
2. 8회초 지루 구간: 이모티콘 4개로 반응 → 리포트에 반영
3. 8회말 파도타기: [점프!] 버튼 → 캐릭터들이 순차 점프
4. 9회말 이민석 타석: 배팅 팝업에서 **홈런** 선택 → 10,000P 배팅
5. 홈런 연출 + 배팅률 그래프 + 더블 적립
6. 경기 종료: 감정 동조율 리포트 + 업데이트된 성향 카드
7. [민지에게 "간맥콜?" 보내기] → 민지 OK → 펍 추천 → 결제 mock

## 안전장치

- Three.js 렌더링 실패 시 ErrorBoundary가 CSS fallback을 보여줍니다.
- 전체 흐름은 Three.js 없이도 끊김 없이 동작합니다.
